Calculation Formulas & Engineering Basis

Total Head

H = (Pd − Ps) × 10.197 / SG
H = Total Head (m) | Pd, Ps = Pressures (bar g) | SG = Density/1000

Hydraulic & Brake Power

Phyd = Q × H × SG / 367   Pbrake = Phyd / η
Q in m³/hr | η = efficiency as fraction

Specific Speed (Ns) — SI Metric

Ns = N × √Q / Hstage0.75
N = RPM | Q = m³/s | Hstage = Head per stage (m)
For double-suction: Qeye = Q/2 (API 610 Annex A)

Suction Specific Speed (Nss)

Nss = N × √Qeye / NPSHr0.75
⚠ Uses NPSHr (NPSH3), NOT NPSHa — API 610 Annex A Eq. A.2

Suction Energy Level (NEW — v4)

SE = SG × N² × (Q/3600)2/3 / NPSHa1/3
SE < 120: Low — OH2 acceptable
120–160: Medium — OH2 with caution
160–200: High — BB2 preferred
> 200: Very High — BB2/BB5 mandatory
Ref: API HI 9.6.1 / Hydraulic Institute

NPSH Margin Check

Margin = NPSHA − NPSHr
Required: Margin ≥ 1.0 m | Recommended: ≥ 1.5 m
Only evaluated when NPSHr is provided. API 610 Cl. 6.1.10 Note 3
