Calculation Formulas & Engineering Basis

Total Head

H = (Pd − Ps) × 10.197 / SG
H = Total Head (m)  |  Pd, Ps = Pressures (bar g)  |  SG = Density/1000

Hydraulic & Brake Power

Phyd = Q × H × SG / 367    Pbrake = Phyd / η
Q in m³/hr  |  η = efficiency as fraction

Specific Speed (Ns) — SI Metric

Ns = N × √Q / Hstage0.75
N = RPM  |  Q = m³/s  |  Hstage = Head per stage (m)
For double-suction: Qeye = Q/2 used for Ns (API 610 Annex A)

Suction Specific Speed (Nss) — API 610

Nss = N × √Qeye / NPSHr0.75
Qeye = Flow per impeller eye (m³/s)  |  NPSHr = NPSHr in metres
⚠ Nss uses NPSHr (NPSH3 from vendor), NOT NPSHa — per API 610 Annex A Eq. A.2
If NPSHr not provided, NPSHa is used as conservative proxy with a warning.

Impeller Tip Speed

u = √(2 × g × Hstage × ψ)
ψ = Head coefficient (function of Ns: 0.4–0.6)
ψ is computed from Ns, not hardcoded, for accuracy across pump types

NPSH Margin Check

Margin = NPSHA − NPSHr
Required: Margin ≥ 1.0 m (minimum)  |  Recommended: ≥ 1.5 m
Only evaluated when NPSHr is provided. API 610 Cl. 6.1.10 Note 3
